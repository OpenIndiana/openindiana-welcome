Bem vindo ao OpenIndiana Hipster!

OpenIndiana é um sistema operacional do tipo Unix baseado no Illumos e derivado do OpenSolaris. Com o objetivo de ser um sistema operacional de propósito geral com funcionalidades de classe enterprise, o OpenIndiana Hipster é desenvolvido pela comunidade, e para a comunidade.

O OpenIndiana obteve este nome a partir do Projeto Indiana, um esforço de código aberto da Sun Microsystems (agora Oracle Corporation) para produzir o OpenSolaris, um sistema operacional Unix-like baseado no Sun Solaris e desenvolvido por uma comunidade. O Projeto Indiana foi liderado por Ian Murdock, o fundador da distribuição Linux Debian.

Algumas das diferenças entre o OpenIndiana e o OpenSolaris podem ser caracterizadas como as seguintes:

  • A consolidação proprietária de software da Sun/Oracle Sun/Oracle OS/NET foi substituída pelo illumos-gate.

  • A maioria do software original do OpenSolaris foi reorganizada em uma consolidação única no oi-userland.

  • O Sun Studio da Sun/Oracle Sun/Oracle foi substituído pelo compilador de código aberto GNU GCC.

  • XVM (XEN) foi substituído pelo port illumos-kvm.

Sobre nós

O Projeto OpenIndiana é uma comunidade formada por voluntários e entusiastas de Unix ao redor do mundo. Atualmente os servidores de build, hospedagem e disponibilidade de banda são doados pela EveryCity no Reino Unido.


Participe

O nosso sucesso como uma distribuição depende do sucesso da nossa comunidade, e é por isso que convidamos todos os interessados no futuro da illumos a participar e se envolver!

Temos muitos recursos a partir do qual você pode começar, incluindo o nosso Wiki, listas de discussão e canal IRC.

  • Para os desenvolvedores, nós possuimos imagens Vagrant de desenvolvimento que podem ser instaladas facilmente.

  • Para usuários finais, nós possuímos servidores públicos de IPS para realizar atualizações de software e instalação de correções de segurança.
