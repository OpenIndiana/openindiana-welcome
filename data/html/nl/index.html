Welkom bij OpenIndiana Hipster!

OpenIndiana is een op illumos gebasseerd Unix-achtige distributie afgesplitst van OpenSolaris. Gebouwd als een algemeen bruikbaar operating systeem met Enterprise Class Features, OpenIndiana is ontwikkeld voor en door de community.

OpenIndiana kreeg zijn naam van Project Indiana, een open source initiatief van Sun Microsystems (tegenwoordig Oracle Corporation) om OpenSolaris te ontwikkelen, een community gedreven Unix-achtige distributie gebasseerd op Sun Solaris. Project Indiana was led by Ian Murdock, founder of the Debian Linux Distribution.

Een aantal van de verschillen tussen OpenIndiana en OpenSolaris:

  • Sun's OS/NET samenvoeging (gesloten door Oracle) is vervangen door illumos-gate.

  • Veel van de originele OpenSolaris software samenvoegingen zijn gereorganiseerd naar een enkele oi-userland samenvoeging.

  • Oracle's Sun Studio is vervangen door de open source GNU GCC compiler.

  • XVM (XEN) is vervangen door de illumos-kvm port.

Over ons

De OpenIndiana Project is een community van vrijwilligers en UNIX enthousiastelingen vanuit de gehele wereld. Op dit moment worden de build servers, hosting en bandbreedte gedoneerd door EveryCity hosting in het VK (UK).


Werk mee

Ons succes als hoogwaardige distributie hangt af van het succes van onze community, daarom verzoeken wij een ieder die geïnteresseerd is in de toekomst van OpenIndiana om te participeren en mee te doen!

We hebben veel bronnen van waaruit je kunt starten, inclusief onze Wiki, Mailing Lijsten en IRC Kanaal.

  • Voor ontwikkelaars hebben we Vagrant ontwikkel images welke eenvoudig kunnen worden geïnstalleerd

  • Voor eindgebruikers hebben we openbare IPS repositories voor het updaten van je installatie naar en met de laatste software en beveiligingspatches.
